Politique de confidentialité

Dernière mise à jour : 17 avril 2025

1. Données collectées

  • ID et pseudonyme Discord
  • Logs de combat fournis volontairement
  • Informations sur les personnages

2. Utilisation des données

Les données sont utilisées uniquement pour fournir les fonctionnalités du bot.

3. Stockage

Les données sont stockées de manière sécurisée. Aucune donnée n'est vendue.

4. Suppression des données

Vous pouvez demander la suppression de vos données en contactant notre support Discord.

5. Modifications

Cette politique peut être mise à jour à tout moment.
